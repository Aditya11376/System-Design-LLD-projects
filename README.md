# System Design - LLD (Java)

This repository contains my practice and implementations of **Low Level Design (LLD)** concepts using Java.

I am using this repo to learn and improve my understanding of:
- Object Oriented Programming (OOP)
- SOLID Principles
- Design Patterns
- Real-world system design problems

### 🔹 Design Patterns
- Factory Pattern
- Singleton Pattern
- Strategy Pattern
- Observer Pattern
- Command Pattern

### 🔹 LLD Projects

- ATM Machine
- Parking Lot System

- Other small design problems
